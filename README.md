Учебный Spring Boot проект с интеграцией базы данных. В нём я активно использовала Hibernate и JPA для работы с данными, Liquibase для миграций, писала юнит- и интеграционные тесты с Testcontainers, проверяла API через Postman, а для работы с БД пользовалась DBeaver. Также настраивала запуск в Docker. Выполняла для понимания работы таких инструментов на практике.
